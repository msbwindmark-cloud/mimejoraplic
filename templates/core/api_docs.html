Endpoints REST

POST /api/sign Requiere API Key

Firma un documento con geolocalización GPS y registro blockchain.

{ "document_id": "uuid-del-documento", "signer_email": "cliente@ejemplo.com", "latitude": 40.416775, "longitude": -3.703790, "liveness_check": true }
Respuesta: hash SHA-256, tx_hash blockchain, certificado descargable.
GET /api/verify/{hash} Público

Verifica la autenticidad de un documento firmado. No requiere autenticación.

GET /api/verify/3f7a8b2c1d9e0f4a5b6c7d8e9f0a1b2c3d4e5f6a
Respuesta: estado, fecha de firma, coordenadas GPS, hash blockchain, firmante.
POST /api/analyze Requiere API Key

Analiza cláusulas de un contrato con IA y devuelve risk score y recomendaciones.

{ "document_id": "uuid-del-documento", "language": "es" }
Respuesta: risk_score (0-100), cláusulas peligrosas, recomendaciones legales.
POST /api/geofence Requiere API Key

Crea una geocerca de protección. La firma solo es válida si el firmante está dentro del área.

{ "document_id": "uuid-del-documento", "latitude": 40.416775, "longitude": -3.703790, "radius_meters": 100 }
Respuesta: geofence_id, estado de protección activa.

Ejemplos de código

curl -X POST https://trustbridge.es/api/sign \ -H "Authorization: Bearer TU_API_KEY" \ -H "Content-Type: application/json" \ -d '{ "document_id": "abc-123", "signer_email": "cliente@ejemplo.com", "latitude": 40.416775, "longitude": -3.703790, "liveness_check": true }'
import requests API_KEY = "tu-api-key" url = "https://trustbridge.es/api/sign" headers = {"Authorization": f"Bearer {API_KEY}"} data = { "document_id": "abc-123", "signer_email": "cliente@ejemplo.com", "latitude": 40.416775, "longitude": -3.703790, "liveness_check": True } resp = requests.post(url, json=data, headers=headers) print(resp.json()) # { "hash": "3f7a...", "tx_hash": "0x...", "status": "SIGNED" }
const API_KEY = "tu-api-key"; fetch("https://trustbridge.es/api/sign", { method: "POST", headers: { "Authorization": `Bearer ${API_KEY}`, "Content-Type": "application/json" }, body: JSON.stringify({ document_id: "abc-123", signer_email: "cliente@ejemplo.com", latitude: 40.416775, longitude: -3.703790, liveness_check: true }) }) .then(r => r.json()) .then(data => console.log(data)); // { hash: "3f7a...", tx_hash: "0x...", status: "SIGNED" }
Integración vía iFrame

Inserta el firmador TrustBridge directamente en tu aplicación. El firmante ve la interfaz de firma sin salir de tu web.

<iframe src="https://trustbridge.es/embed/sign?token=TOKEN_UNICO" width="100%" height="600" frameborder="0" allow="geolocation; camera"> </iframe>
  • Solicita permisos de geolocalización al usuario
  • Captura liveness con la cámara (opcional)
  • Callback vía postMessage cuando la firma se completa
